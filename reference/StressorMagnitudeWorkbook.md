# Get Stressor Magnitude Data from Excel Workbook

Extract the stressor magnitude data from each sheet in the Excel
workbook.

## Usage

``` r
StressorMagnitudeWorkbook(filename = NA, scenario_worksheet = NA)
```

## Arguments

- filename:

  A string. Relative file name to the stressor magnitude Excel workbook.

- scenario_worksheet:

  A string. Worksheet name for the scenario.

## Value

A stressor magnitude list object for the CEMPRA

## Details

For more information about data format requirements, see the Data Upload
section of the vignette.
